為什麼需要 Gen3？

Gen2 使用 American Airlines 私有 Snowflake 資料庫，無法公開重現——這在學術論文中是重大限制。Gen3 的目標是：

在完全公開的資料上，用相同的方法論框架，得到可重現且可被審查的結果。

12 條洩漏控制規則

Gen3 為每個特徵設計了明確的可用性審計規則，確保所有特徵在 T-2h 到 T-6h 前均可取得：

Tip

34/34 特徵全部通過 12-Rule Leakage Checklist ✅

排班特徵：起飛前數小時/數天已知，全部安全 機場靜態：長期穩定數據，安全 日曆特徵：天文事實，安全 METAR 天氣：merge_asof(backward) 確保只用觀測時間 ≤ 截止時間的資料 TAF 預報：使用預報時間 ≤ T-2h 且預報效期涵蓋到達時刻的資料 尾鏈特徵：以實際到達時間判斷前班是否可知（同 V9c 修正）

V4 vs V2 差距分析

Tip

V4（Clean Baseline）：AUC = 0.7011 V2（Full Pipeline）：AUC = 0.7681 差距：−0.0670

這個差距量化了以下四個因素的合計貢獻： 1. Tail Chain 特徵（6 個） 2. Congestion 特徵（3 個） 3. Optuna HPO 4. SHAP 特徵篩選

V4 的 SHAP 分析（CatBoost，50K test 樣本）提供了在「沒有尾鏈和壅塞」條件下的特徵重要性基準。

設計

V2 在 V4 的 34 個乾淨特徵上，加入： - 尾鏈特徵（tail chain，6 個） - 壅塞特徵（congestion，3 個） - SHAP backward elimination 特徵篩選（41 → 41，選擇結果：無特徵被移除，全部保留） - Optuna TPE Bayesian HPO（100 trials）

Optuna 最佳結果

Subsample Val AUC: 0.7559（2M 訓練樣本子集） Full-train Val AUC: 0.7642（完整 12.4M 訓練集驗證） 最佳迭代次數：4995 HPO 時間：3238 秒（54.0 分鐘）

核心發現

Tip

「看排班就知道會不會延誤」

在嚴格 leakage control 下，排班衍生特徵（dep_hour、dep_month、is_holiday_period 等）系統性地主導了預測。延誤有超過一半的可預測部分，在排班表確定的那一刻就已決定了。

這意味著什麼

延誤不是隨機意外，而是結構性問題。 有些時段（早班 vs 晚班）、有些月份（假日前後）、有些路線，在排班確定時就已預設了更高的延誤風險。

Gen2 ablation 數據支持此結論： - Schedule-only 基準：AUC = 0.6226 - 加入全部特徵（43個）：AUC = 0.7279 - Schedule 特徵解釋了 ~56% 的可預測變異（0.6226 / 0.7279）

文獻中的缺口

沒有任何論文曾在嚴格 leakage control 下，用 SHAP 量化「排班主導的比例是多少、為什麼」。

FDPP-ML（Journal of Big Data, 2023）曾指出可以只靠排班特徵預測延誤，但未解釋為什麼，也未量化比例。

我們在 19.15M 航班規模上嚴格驗證，不是地理特例，而是系統性結構發現。

對航空業的意義

如果延誤主要由排班結構決定，那麼： - 航班調度優化比即時干預更有效 - 高延誤機率時段的過密排班是根本問題 - 「提前預測」的核心價值在於讓乘客和運營商早做安排，而非「在延誤發生時才知道」

核心發現

Tip

在嚴格 pre-departure leakage control 下，天氣特徵的 SHAP 貢獻在 Gen3 中接近零（相對於尾鏈和排班）。

這與大部分文獻的結論相反——但我們認為這是方法論差異，而非真實世界的差異。

為什麼天氣在我們的模型中不重要？

物理原因： - 起飛前 2–6 小時的 METAR 天氣觀測代表「現在」的天氣，不是航班到達時的天氣 - 長程航班（如 DFW → JFK，飛行時間 3–4 小時）在起飛時的目的地天氣預測精度有限 - TAF 預報的時間分辨率（典型預報間隔 1–6 小時）使預報與實際到達時天氣仍有差距

Warning

現有文獻中天氣「重要」的可能原因

許多論文在計算天氣特徵時，使用了起飛後或到達後的天氣觀測值（實際離港時間、到達時的 METAR），這構成資訊洩漏。在嚴格的 T-2h 截止條件下，這些「未來天氣」無法取得，天氣的表觀重要性大幅下降。

Tip

我們的貢獻：美國天氣複雜多變，天氣「仍然不重要」只能由方法論解釋——在嚴格 pre-departure leakage control 下，天氣的表觀重要性可能主要是測量偏誤的產物。
