資料科學與資訊計算研究所 · 國立中興大學

學歷： 理學碩士，國立中興大學資料科學與資訊計算研究所（Master of Science in Data Science and Information Computing, NCHU）。

關於本所

資料科學與資訊計算研究所隸屬於國立中興大學理學院，成立宗旨為培育兼具資料科學理論基礎與資訊計算實作能力的跨領域人才。課程涵蓋機器學習、深度學習、大數據分析、影像處理、最佳化方法及高效能計算，強調「從數學到實作」的完整訓練流程。

本所整合數學基礎與現代計算技術，培養機器學習、深度學習、電腦視覺及大數據分析領域之專業人才。學生將具備連結理論模型與實際應用的能力，應用領域涵蓋醫療、工業及科學研究等。

研究重點領域

顏色圖例： 綠色 = 機器學習、橘色 = 深度學習、紫色 = 電腦視覺、粉色 = 最佳化。以上為課程涵蓋的主要研究領域。

習得技能

技能來源： 粉色 = 機器學習與深度學習課程共同訓練的技能、藍綠色 = 主要來自機器學習課程、橘色 = 主要來自深度學習課程。Python/PyTorch 為兩門課程共用的程式語言，遷移學習（Transfer Learning）與模型評估（Model Evaluation）也在課程中反覆練習。

修課內容

機器學習與資料科學（Machine Learning & Data Science）

課程重點： 監督式／非監督式學習、模型評估、特徵工程、資料前處理

作業：

  • HW1 – 糖尿病預測： Pima Indians 糖尿病預測。迴歸填補 -> 特徵工程 -> DNN，從基線 74% -> 90.04%
  • HW2 – 美國野火分析： 188 萬筆野火紀錄。Poisson 迴歸趨勢分析 + MLP 成因分類（45.6%）
  • 期末 – 子宮頸癌篩檢： EfficientNet-B7 遷移學習 + Focal Loss，三分類影像分類，平均 86.1%

主要工具： Python · Keras · scikit-learn · statsmodels · PyTorch（期末）

深度學習（Deep Learning）

課程重點： CNN 架構、遷移學習、語意分割、影像生成、自編碼器

作業：

  • HW1 – AOI 瑕疵分類： ResNet-18 遷移學習，6 類工業瑕疵分類，96.44% Val Acc
  • HW2 – 視網膜血管分割： U-Net（5 層）+ Focal Tversky Loss，DRIVE 資料集，mIoU 0.351
  • HW3 – 視網膜影像重建： 卷積自編碼器，峰值 PSNR 30.84 dB（Epoch 18）
  • HW4 – Western Blot 生成： 條件式 GAN（Generator + PatchGAN Discriminator），分析 D/G 訓練動態

主要工具： Python · PyTorch · torchvision · Apple Silicon (MPS)

大數據分析（Big Data Analysis）

課程重點： 核方法加速、大規模最佳化、分散式機器學習

作業：

  • 文獻閱讀 – Nystrom 方法： 論文閱讀（NIPS 2000），Gram 矩陣低秩近似，O(n^3) -> O(m^2n)
  • HW – Kernel Ridge + Nystrom： USPS 手寫數字分類，m=128 達成 20 倍加速，準確率 99.50%
  • 期末 – Smoothed & Distributed SVM： a9a 資料集，Smoothed Hinge Loss + 分散式梯度聚合（K=5 workers），150 倍加速

主要工具： Python · NumPy · scikit-learn · SciPy (L-BFGS-B)

資料分析數學（Mathematics for Data Analysis）

課程重點： SVD 理論與應用、低秩矩陣近似、Eckart-Young 定理、手寫數字辨識

作業：

  • HW1 – SVD 影像壓縮： 以照片驗證 Eckart-Young 定理，Monte Carlo 近似 2-norm，PSNR 達 44.7 dB（k=700）
  • HW2 – 手寫數字辨識： USPS 資料集，比較 8 種方法（Mean / SVD / HOSVD / SVM / KNN / RF / CNN），CNN 95.76% 最高

主要工具： Python · NumPy · PyTorch · tensorly · scikit-learn
